En primer lugar, descargar el código fuente contenido en el archivo FisFinder.tar.gz.
Luego extraer los datos del archivador donde se desee. Una vez realizados los pasos anteriores, continuar con las siguientes instrucciones:

Para correr el programa, primeramente debe instalarse "Flask". Para ello, desde el terminal debe ejecutarse el comando:

pip install flask

Posteriormente, se debe estar en la carpeta Vista y ejecutar el progrma server.py. En la linea de comando se ejecuta

python server.py (mantener en ejecución)

Iniciar navegador WEB preferido e ir a localhost:5000. Tras realizar el paso anterior se cargará el prototipo del sistema.
 
En la barra dispuesta para ingresar texto escriba lo que desee buscar, tras lo anterior deberá presionar el botón buscar.

Podrá visualizar los resultados de la búsqueda. Para explorar alguno pinche su título.
